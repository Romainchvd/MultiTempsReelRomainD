# MultiTempsReelRomainD

Initialisation:

Afin de jouer, il faut qu'un serveur soit disponible:
- l'un des joueurs doit appuyer sur Host Session pour lancer une partie
- L'autre joueur doit rechercher une session et se connecter au serveur désiré

Lancement de la partie:

Pour lancer une partie, il faut que tous les joueurs présents soient prêts:
tous les joueurs doivent appuyer sur la touche G pour toggle leur statut.
Lorsque tous les joueurs sont prêts, un compte à rebours commence.
Le compte à rebours sera annulé si:
- L'un des joueurs n'est plus prêt
- Un joueur vient de rejoindre la partie
Il est possible de lancer la partie en étant seul : le deuxième joueur apparaîtra alors directement en jeu.
